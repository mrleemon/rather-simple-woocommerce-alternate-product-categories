== Description ==

A really simple WooCommerce alternate product categories widget and block.

This plugin is experimental and is provided with no support or warranty. 
This plugin requires WooCommerce.

== Installation ==
1. Make sure you have installed and activated the latest version of WooCommerce
2. Upload the extracted plugin folder and contained files to your /wp-content/plugins/ directory
3. Activate the plugin through the "Plugins" menu in WordPress
